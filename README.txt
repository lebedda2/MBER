WHAT'S HERE:
-------------
In this directory, there are several program nodes that operate the MBER
project.  If the node is labelled with remote_* then this node operates on the
remote system.  If the node is labelled with mber_* then this node operates on
the robot itself.  At the moment, the specific nodes are:
  >> (remote) Remote Control Module w/ PS3 Controller
  >> (remote) GUI Control
  >> (mber)   Motor Controller
  >> (mber)   Sensor Controller
  >> (mber)   Camera Controller
